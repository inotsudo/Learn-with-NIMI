# Story Assets — "The Talking Faces"

Drop your files into these folders, then run 003_seed_story_content.sql in Supabase.

## /pages/
Illustrated story page images, one per page.
Name them: page-1.jpg, page-2.jpg, page-3.jpg ... (JPEG or PNG, portrait preferred)

## /audio/
Narration audio for each page — Nimi reads the text aloud.
Name them: page-1.mp3, page-2.mp3, page-3.mp3 ... (MP3)

## /coloring/
Black-and-white line drawing versions of the pages for the Little Creators coloring step.
Name them: page-1.jpg, page-2.jpg, page-3.jpg ... (JPEG or PNG)
If you don't have separate coloring versions, just copy the same images from /pages/ here.

## Root of this folder
- journey.mp4  → the animated story video for Step 6 (Journey with Nimi)
- song.mp3     → the sing-along song for Step 5 (already have morning-song.mp3 — copy it here or use a new one)
