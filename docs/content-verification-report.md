## 7. Recommendations for BK.4B.2 (Unit 2 onward)

1. **Reuse Zara and Amara** as recurring characters across Unit 1–4 (Block 1). Introducing
   the same characters across a block builds familiarity — new characters can be introduced
   in Block 2.
2. **CMS import sequence:** Import all three language versions of one lesson before moving
   to the next lesson — this avoids orphaned EN-only missions sitting in the CMS during
   authoring.
3. **Song melody notes:** Future Morning Song tip texts should include a tempo descriptor
   (e.g. "Sing to a steady 4/4 beat, approx. 80 BPM") to give audio producers a reference
   when recording songs.
4. **Media URL placeholder:** None of the 8 lessons have a `media_url` yet. This is correct
   for BK.4B — media assets are BK.4C scope. Leave `media_url` null on import.
5. **Kinyarwanda review:** Kinyarwanda content should be reviewed by a native speaker before
   moving status from `review` to `published`. Flag this in the CMS workflow.
