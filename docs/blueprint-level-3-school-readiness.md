# Level 3 — School Readiness Curriculum Blueprint

**Age Range:** 5 – 6 years
**Framework:** Pre-Literacy, Pre-Numeracy & Critical Thinking
**Total Units:** 52 | **Blocks:** 13 × 4 units
**Languages:** English · Français · Kinyarwanda

## Guiding Principles
- Phonics and early reading embedded across all categories
- Pre-numeracy: counting to 100, addition/subtraction within 10, measurement, data recording
- Scientific inquiry: hypothesis → experiment → observation → conclusion
- Critical thinking: compare past and present, evaluate, explain reasoning
- Cultural identity deepened: Rwanda's history, geography, civic values, and language as analytical lenses
- School-readiness behaviours: turn-taking, listening, independent task completion, written response

## Complexity Profile
| Category | School Readiness Approach |
|---|---|
| 🎵 Morning Song | Phonics-infused song with letter sounds, rhyme patterns, and syllable clapping |
| 🏃 Movement Mission | Multi-step sequence with embedded counting, spatial reasoning, and label-reading |
| 🎨 Mission Artistique | Plan → create → label: complex technique with a written or dictated description |
| 🏛️ Mission Historique | Read or listen to a text; compare past/present; write or dictate a response |
| 🔍 Mission Zoom | Scientific observation: record findings in a table or drawing, form a hypothesis |
| 🌱 Mission Discovery | Full inquiry cycle: predict → experiment → measure → record numbers → conclude |
| 📖 FlipFlop Book | Read independently or with support; identify narrative structure; write a response |
| 🖍️ Coloring Book | Complex scene with deliberate colour strategy, pattern, and optional written label |

## Block 1 — Identity & Welcome (Units 1–4)

### Unit 1 — Hello, World!
**Development Focus:** Trilingual identity, phonics of greetings, and school community belonging
**Learning Outcomes:**
1. Read and write the greeting words "Hello," "Bonjour," and "Muraho" independently
2. Explain why greetings matter using at least two reasons
3. Compose a two-sentence self-introduction in writing or dictation

| Category | Objective |
|---|---|
| 🎵 Morning Song | Sing a greeting song rich in initial sounds — /h/, /b/, /m/ — and clap syllables in each greeting word |
| 🏃 Movement Mission | Follow a five-step greeting routine (read the action cards: bow, shake, wave, smile, say name) |
| 🎨 Mission Artistique | Plan a "My Hello" card: sketch, choose colours deliberately, write the greeting word inside |
| 🏛️ Mission Historique | Read a short text about why Rwandans greet elders first; compare to their own family practice |
| 🔍 Mission Zoom | Observe and record: in a table, list greeting words heard in three videos (EN/FR/RW), note similarities |
| 🌱 Mission Discovery | Hypothesis: "Which greeting do I know best?" — experiment by timing recall, record results, conclude |
| 📖 FlipFlop Book | Read "Good Morning, Rwanda" independently; identify the problem, solution, and moral |
| 🖍️ Coloring Book | Colour a sunrise over Kigali with children greeting; label each child's speech bubble |

### Unit 2 — My Name Is…
**Development Focus:** Name literacy, letter formation, and personal identity narrative
**Learning Outcomes:**
1. Write their full name (first and family) correctly with consistent letter formation
2. Identify the first letter of their name and list three other words that start with the same sound
3. Write or dictate three sentences about what their name means to their family

| Category | Objective |
|---|---|
| 🎵 Morning Song | Sing "My Name Starts With…" — a phonics song where each verse spotlights one letter sound |
| 🏃 Movement Mission | Write their name in the air, on the ground, and on a partner's back; compare how it feels |
| 🎨 Mission Artistique | Design an illuminated name initial: plan the design, use pattern fills, label the colours used |
| 🏛️ Mission Historique | Read a text about Kinyarwanda naming traditions; write one sentence about their own name's story |
| 🔍 Mission Zoom | Observe a chart of the most common names in Rwanda; read and record the top three names found |
| 🌱 Mission Discovery | Measure their name: count letters, count syllables, record both numbers; compare with five classmates |
| 📖 FlipFlop Book | Read "My Name Is My Story"; write a response — one thing they learned about their own name |
| 🖍️ Coloring Book | Colour an intricate name-banner; write or trace their name below it and add a personal symbol |

### Unit 3 — My Five Senses
**Development Focus:** Sensory vocabulary, descriptive writing, and scientific observation
**Learning Outcomes:**
1. Write a descriptive sentence for each of the five senses using adjectives
2. Classify ten objects by two sensory attributes simultaneously
3. Design and record a simple sensory experiment with a prediction and a conclusion

| Category | Objective |
|---|---|
| 🎵 Morning Song | Sing a sensory adjective song; clap syllables in "silky," "crunchy," "fragrant," "bitter," "shrill" |
| 🏃 Movement Mission | Follow written sensory-challenge cards: "Find something rough, cold, and small" — complete each step |
| 🎨 Mission Artistique | Create a "Sensory Rwanda" artwork; plan colours to represent each sense, label each section |
| 🏛️ Mission Historique | Read a story set in a Rwandan market; write two sentences describing the senses the character used |
| 🔍 Mission Zoom | Observe and record in a five-column table: one sensory word per sense for each of three objects |
| 🌱 Mission Discovery | Predict which sense is hardest to describe; conduct a timed description test; record and conclude |
| 📖 FlipFlop Book | Read "The Sense Detectives"; identify the narrative structure (beginning/middle/end) and write a summary |
| 🖍️ Coloring Book | Colour a Rwandan market scene; write one sensory label on five different items in the picture |

### Unit 4 — I Am Unique
**Development Focus:** Self-concept, comparison, and expressive writing
**Learning Outcomes:**
1. Write a five-sentence "All About Me" paragraph using a planning frame
2. Name two things that make them different from a peer and explain why difference is valuable
3. Read their paragraph aloud with expression and confidence

| Category | Objective |
|---|---|
| 🎵 Morning Song | Sing "I Am One of a Kind" — a song with rhyming pairs that celebrate individual traits |
| 🏃 Movement Mission | Complete a "unique skills" circuit: each station has a card with a different challenge to attempt |
| 🎨 Mission Artistique | Create a self-portrait with a written "artist's statement" — three words that describe them |
| 🏛️ Mission Historique | Read a text about Rwanda's value of ubumuntu (shared humanity); discuss: how can we be alike and unique? |
| 🔍 Mission Zoom | Observe fingerprints under a magnifier; record: what patterns do you see? Are any the same? |
| 🌱 Mission Discovery | Measure height, hand span, and foot length; record in a table; compare data with three peers |
| 📖 FlipFlop Book | Read "Nobody Else Like Me"; write a response: "One way I am unique is…" (two sentences) |
| 🖍️ Coloring Book | Colour a mirror-frame template; draw their own face inside and label five unique features |
